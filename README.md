# Synchrono City

> *Talk first, meet later.*

Drop a pin anywhere on the map to start a group chat: text and voice/video. Join from anywhere. If you vibe, take it to the DMs. Meet up in person, maybe. Your location stays private. Your DMs, end-to-end encrypted. *No swiping, just talking.*

---

Open source, built on Nostr and LiveKit.
